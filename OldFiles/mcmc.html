Markov Chain Monte Carlo

The learning objectives for this topic:
  • Implement Gibbs sampling algorithm, a random-walk Metropolis algorithm, a Metropolis-Hastings algorith, and a "Metropolis-in-Gibbs" algorithm
  • Implement MCMC using R, WinBUGS/OpenBUGS, and SAS
  • Apply MCMC diagnostics
